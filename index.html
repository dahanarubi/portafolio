PORTAFOLIO DE EVIDENCIAS

Este portafolio documenta el proceso de aprendizaje y desarrollo de aplicaciones web orientadas a servicios, mostrando las competencias adquiridas en la integración de tecnologías frontend y backend, así como la aplicación de buenas prácticas de programación, incluyendo la creación e integración de APIs y su despliegue en un hosting real. reflejando el crecimiento técnico y práctico durante el cuatrimestre.
Sus objetivos principales son consolidar conocimiento y presentar evidencias de las actividades realizadas en cada unidad.

Unidades de Materia

Cada unidad representa una etapa del aprendizaje, desde los fundamentos teóricos hasta la implementación práctica de servicios web.

UNIDAD I
Introducción al DWOS

Conceptos básicos y arquitectura SOA

Se estudian los fundamentos del desarrollo orientado a servicios y la arquitectura SOA.

UNIDAD II
Implementación de API

Uso e integración de APIs

Consumo de APIs REST, manejo de solicitudes HTTP y pruebas con Postman.

UNIDAD III
Desarrollo de API

Creación de servicios

Diseño y desarrollo de APIs propias aplicando buenas prácticas.

UNIDAD IV
Implementación

Aplicación práctica

Integración completa de servicios en una aplicación funcional.

Metodología y Aprendizaje

Aprenderemos a desarrollar aplicaciones web orientadas a servicios, integrando APIs, manejando bases de datos, aplicando buenas prácticas de programación y publicando proyectos en hosting real, reforzando competencias técnicas y prácticas en desarrollo web.

El desarrollo del portafolio se realizó siguiendo una metodología incremental, permitiendo reforzar progresivamente los conocimientos adquiridos.
